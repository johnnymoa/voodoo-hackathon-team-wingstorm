# reference/

Canonical playables to study. Not generated, not consumed by pipelines —
they're here so you can read what "good" looks like before tweaking
`src/adforge/templates/playable_template.html` or the `build_playable_html`
activity.

| File              | Why it's here                                                   |
|-------------------|------------------------------------------------------------------|
| `MarbleSort.html` | Clean single-file CONFIG-block pattern, minimal dependencies.   |
| `CupHeroes.html`  | Richer playable with sprites + sound, useful as a stretch goal. |
